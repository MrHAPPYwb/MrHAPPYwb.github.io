# 研姐的宝石任务岛

面向小学一年级儿童的竖屏 PWA 学习手游，可在 iPhone Safari 中使用并添加到主屏幕。

## 内容

- 语文 22 关：象形识字、规范笔顺、拼音、偏旁、量词、反义词、组词、句子、标点、阅读、古诗和口语表达。
- 数学 20 关：0—20 数的认识、比较、组成、加减法、图形、位置、整时、人民币、分类、规律、数据和生活购物。
- 英语 24 关：覆盖一二年级常见的动物、食物、颜色、身体、家人、学校、交通、天气、服装、时间、方位和动作主题。
- 绘画花园：真实触控画布、颜色、画笔、橡皮和本地作品存档。
- 宝石工坊：将语文命名、数学配比、英语拼词和手指抛光融合成完整制造流程。

中文 NPC 优先选择设备上的甜美女声；英语优先选择标准美式女声，并在每次英文播题后继续播报中文题意。

## 开发

```bash
npm install
npm run dev
npm run build
npm run lint
```
