# 研姐的水晶矿山

面向小学一年级儿童的竖屏 PWA 学习手游，以黄金矿工的抓取循环承载语文、数学和英语练习。

## 游戏系统

- 钩子持续摆动，点击矿洞发射，抓取使用 Three.js 离屏渲染的立体彩钻。
- 钻石拉回后打开知识晶核；即时反馈解释答案，而不只报告对错。
- 100 个关卡、300 道基础变式题：40 关语文、40 关数学、20 关英语。
- 错题至少间隔两个关卡后重新出现，形成检索练习和间隔复习。
- 每 5 关赠送一颗 3D 彩钻，共 20 次奖励。
- 集齐玫红、海蓝、翡翠、紫晶和蜜糖五色钻石后进入锻造工坊。
- 在 Three.js 3D 场景中敲打钻石，锻造成水晶小兔、小猫、公主冠或蝴蝶。
- 进度和作品保存在设备本地，无账号、广告、付费和 WebSocket。

课程以教育部 2022 年义务教育课程标准为边界，参考人教版一年级教材目录，同时兼顾主流版本共有的一年级知识范围。

## 开发

```bash
npm install
npm run dev
npm run build
npm run lint
```

生产版本通过 GitHub Pages 以普通 HTTPS 静态文件发布。
