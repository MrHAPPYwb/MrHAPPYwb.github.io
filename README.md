# 研姐的水晶矿山

面向小学一年级儿童的竖屏 PWA 学习手游，以黄金矿工的抓取循环承载语文、数学和英语练习。

## 游戏系统

- 钩子持续摆动，点击矿洞发射；绳索先快后慢并带阻尼摆动，抓手采用玫瑰金金属和钻石材质。
- 矿场固定放置 9 颗五色大小不一的 3D 知识钻石、3 块发光矿石障碍和 1 个水晶宝箱。
- 钻石拉回后打开知识晶核；即时反馈解释答案，而不只报告对错。
- 100 个关卡、300 道基础变式题；每关固定语文 3 题、数学 3 题、英语 3 题，抓齐 9 颗宝石才通关。
- 错题至少间隔两个关卡后重新出现，形成检索练习和间隔复习。
- 题目语音使用随游戏打包的神经女声：普通话 `zh-CN-XiaoxiaoNeural`，美式英语 `en-US-AnaNeural`，不调用设备 TTS。
- 每关有一个 3D 宝箱，可获得星光贴纸、锻造星火或时间水晶；拉回后暂停游戏并展示用途说明。
- 每通过一关赠送一颗彩钻，共 100 次奖励。
- 集齐玫红、海蓝、翡翠、紫晶和蜜糖五色钻石后进入锻造工坊。
- 在 Three.js 3D 场景中挥动实体锻造锤，配合金属撞击、火花、震动和逐步成形反馈。
- 可锻造小兔、小猫、公主冠、蝴蝶、心愿宝石、星辰徽章、水晶城堡和晨露晶花共 8 种作品。
- 锻造完成后进入成果揭幕，作品永久保存在可旋转查看的水晶陈列馆中。
- 进度和结构化作品记录保存在设备本地，无账号、广告和付费；运行时不依赖实时连接。

课程以教育部 2022 年义务教育课程标准为边界，参考人教版一年级教材目录，同时兼顾主流版本共有的一年级知识范围。

